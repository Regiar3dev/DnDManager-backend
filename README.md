# DnDManager-backend

## Descripcion

Este proyecto busca armar una API backend utilizando Node.JS, TypeScript y Express para la WebApp "DnDManager".

Esta API proveera los endpoints para:
- Logeo de usuarios
- Creacion de campañas
- Creacion de Sesiones por campaña
- Creacion de personajes por campaña
- Actualizaciones de estado de personajes por sesion

A su vez, se comunica con el servicio de API de "DnD5e" para fetchear los datos necesarios al momento de crear los personajes (Raza, Clase, Estadisticas, Traits, Habilidades, Items y Descripciones)
